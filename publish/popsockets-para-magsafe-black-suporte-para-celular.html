# Descrição do produto


Um preto incrivelmente ousado, este PopGrip + para MagSafe se conecta facilmente à sua capa compatível com MagSafe para oferecer uma maneira elegante e segura de transportar seus cartões no telefone. PopGrip + para MagSafe é melhor usado quando emparelhado com um case compatível com MagSafe.


                                                                       

Removível

A carteira é facilmente removida do telefone para acesso rápido com cartão

                                                                     

Compatível com montagem

Basta encaixar o punho em um suporte PopSockets para usar o telefone com as mãos livres

Detalhes e especificações
  • Pega e suporte magnético para telefone com topo trocável
  • Ímãs embutidos se conectam com segurança ao estojo compatível com MagSafe
  • Permite o uso mãos-livres com os PopMounts. Compatível com todas as montagens vendidas hoje
  • Basta deslizar a alça para carregar MagSafe
  • Dimensões: Largura da base: 56,90 mm, Comprimento da base 90,00 mm, Espessura da base e do PopTop 7,12 mm
Verificação de compatibilidade

Funciona com:

  • Capas de telefone compatíveis com MagSafe
  • Compatível com todos os produtos PopMount, para visualização em mãos-livres
  • Compatível com todos os PopTops

Não funciona com:

  • iPhone 11 ou designs anteriores
  • Capas de telefone não compatíveis com Magsafe

PopGrip para Magsafe se conecta com segurança ao seu case compatível com MagSafe para máxima aderência e utilidade do suporte, enquanto simplesmente desliza (quando você quiser) para carregar MagSafe.

Oferece uma pegada segura para que você possa enviar mensagens de texto com uma mão, tirar melhores fotos e assistir a vídeos de gatos com as mãos livres. Funciona como um suporte conveniente para que você possa assistir a vídeos rapidamente.

Use o seu telefone mãos-livres. Use com um carro, mesa, parede ou montagem flexível. Criar conteúdo de vídeo ficou muito mais fácil.
